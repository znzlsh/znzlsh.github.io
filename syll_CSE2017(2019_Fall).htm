CSE 2017-03 Data Structure and Lab.

Fall 2019

 

This course is an introductory course in data structures. Topics include arrays, stacks, queues, linked lists, recursion and binary trees. The student will build projects using the data structures taught in class. Proper software engineering techniques are introduced, including object oriented design and programming and unit testing.
